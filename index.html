🧠 AI Rules hard overrides that always apply

Rules are injected into every prediction with higher priority than historical data. Use them to correct outliers, flag supply constraints, or lock in known truths about your markets.

💡 Examples
"Neptune Festival always performs above average regardless of rain — regulars show up"
"Pikachu demand always exceeds my supply — never underestimate it"
"Ignore Kara booth events when benchmarking revenue — different operator dynamic"
"VB Farmers Market in winter averages $40 less than other seasons"
📊 Google Sheets — Piece Rate Sync

Connect each contractor's Google Sheet to auto-refresh their piece rates. Publish the sheet as CSV first: File → Share → Publish to web → CSV. Rates refresh every time you open the Pay tab.

📋 Your Sheet ID is the long string in the URL:
docs.google.com/spreadsheets/d/[SHEET ID HERE]/edit
📤 Order Log — Google Sheet Webhook

Paste your Google Apps Script web app URL here. Every time you save or pay an order, it auto-logs to your pink Google Sheet. See setup instructions below.

Setup (one time):
1. Download the Amioki Order Log.gs file below
2. Open script.google.com → New Project
3. Paste the script → Save → Deploy → New deployment → Web app
4. Set "Who has access" to Anyone → Deploy → Copy URL → paste above
